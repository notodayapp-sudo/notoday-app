Notoday

Notoday is a responsive web landing page built to support individuals who want to stop gambling.
The project is currently in MVP stage and focuses on validating user interest through a waitlist system.

Overview

This project was developed as the initial version of a future web/mobile application aimed at behavioral recovery support.

The current implementation includes:

Multi-language structure (PT / EN)

Email waitlist integration

Google Analytics tracking

Responsive layout

Deployment via Vercel

Tech Stack

HTML5

CSS3

Vanilla JavaScript

Google Analytics (gtag.js)

Vercel (Hosting & CI/CD)

Architecture

The project follows a simple static structure:

/pt          → Portuguese version
index.html   → English version
/assets      → Images and static files
style.css    → Global styling
script.js    → Frontend logic


Language redirection is handled based on browser settings.

Deployment

Hosted on Vercel with automatic deployment connected to GitHub repository.
